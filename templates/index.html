About

MenuQR is a free easy to use tool to help you generate a QR code to digitize your menu. Provide a link to your menu, print the code and you are good to go!

Benefits include less spread of Covid-19 as hardback menus are not shared among customers. Also save on printing costs! One Qr code will cover all your menu updates!

There is no cost, please try it out for free! If you are happy that it works to your expectation, a small tip would be helpful to cover development and running costs.

Check out the examples below. The QR code can link to your website or a drive folder where your menu is stored.

Click the button to generate your code with instructions.
